Install:
1. Unpack the compressed package
2. Copy all files to the \plugins\LiteLuaLoader directory
3. Check whether iland-core.lua is in the \LiteLuaLoader\lua directory
4. Open server test

Update:
1. Unpack the compressed package
2. Delete config.json, data.json, owners.json in the data\iland directory in the decompression result
3. Repeat the "Install" step 2-4
